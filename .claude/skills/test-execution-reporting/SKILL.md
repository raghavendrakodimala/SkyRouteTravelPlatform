# Test Execution Reporting Skill

Test summaries must include:

- Branch
- Commit hash if available
- Environment
- Commands executed
- Results by test area
- Failed tests
- Evidence
- Defects
- Risks
- QA recommendation

Store under:

- `docs/testing/execution/`

Never claim tests passed without command output.

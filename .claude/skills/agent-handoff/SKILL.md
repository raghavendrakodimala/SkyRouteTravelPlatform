# Agent Handoff Skill

Every handoff must include:

- Handoff ID
- Date
- Branch
- Phase
- From agent
- To agent
- Status
- Work completed
- Artifacts created or updated
- Decisions made
- Open questions
- Risks and impediments
- Required next action
- Relevant files

Store handoffs under:

- `docs/handoffs/`
